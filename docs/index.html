Welcome to Risk in a Box’s documentation!¶

Risk in a Box is a Quantum GIS plugin that enables you to perform risk scenario analysis of natural disaster events such as a flood, tsunami, earthquake etc.

It is developed jointly by AusAID (AIFDR) and World Bank (GFDRR) as an Open Source project

You can find out more about the Risk in a Box project by visiting riskinabox.org.
